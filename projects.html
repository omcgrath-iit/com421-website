Assembly RPN Calculator

A calculator program written in MIPS32 assembly code for SPIM-like simulators. It is similar in concept to the Unix program dc, though with a far simpler interface and implementation. Expressions are input in reverse Polish notation, i.e, operators appear after their respective operands; see documentation for further details. Supported operations, for integral arithmetic on 32-bit integers, include:

  • Addition
  • Subtraction
  • Multiplication
  • Division
  • Modulo
  • Summation
  • Exponents
  • Factorials
  • Conversion from binary to decimal
  • Control operations for managing the RPN stack

I/O Benchmark Program

A program for benchmarking file I/O on Linux. It can test latency and throughput in sequential writes, sequential reads, random writes, and random reads in single-threaded or concurrent workloads. It uses direct I/O to ensure it is measuring disk speed, not cache speed.

The source code is written in C and uses only C standard libraries and Linux system libraries. Included are shell and AWK scripts for automated testing of many configurations; running a similar benchmark program (IOzone) and formatting its results; and generating reports comparing each program's output in equivalent tests.

XMonad Extensions

Assorted extensions for the XMonad window manager. Written in Haskell, built with Cabal, and can be installed via Nix flakes. Some of the most interesting modules include:

XMonad.Actions.CycleSettings
Allows cycling between various sets of settings (e.g., spacing and decorations), on a per-workspace basis
XMonad.Hooks.DisableShadows
For use with a compositor, can set appropriate windows properties to disable shadows where they do not make sense
XMonad.Layouts.Toggle3Layouts
A layout similar to XMonad.Layout.ToggleLayouts except that it allows toggling between three layouts
